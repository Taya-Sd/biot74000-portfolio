_An honest summary of my current bioinformatics competencies._

## Intermediate / Advanced

_Skills I can perform independently, explain to others, and apply in a new situation._

| Skill / Tool | What I can do with it |
|---|---|
| NCBI Gene lookup | Find a gene record and confirm its gene name, organism, Gene ID, chromosome location, and links to related records. |
| NCBI RefSeq sequence retrieval | Locate a gene within a reference chromosome, check its coordinates and strand, and retrieve only the required gene sequence. |
| FASTA sequence handling | Recognize the FASTA structure, check the header and sequence length, and prepare a nucleotide sequence for use in another tool. |
| Basic BLASTN searches | Enter a nucleotide query, select a nucleotide database, run the search, and review the main results. |
| Basic BLAST interpretation | Compare results using E-value, percent identity, query coverage, and bit score and recognize full-length and partial matches. |
| Database record comparison | Compare NCBI and UniProt records based on sequence information, protein function, evidence, annotation quality, and reliability. |
| Analysis documentation | Record accession numbers, databases, search settings, dates, screenshots, and important decisions so the work can be repeated. |

## Beginner / Basic Exposure

_Skills I have used in a guided setting and would need support to apply independently._

| Skill / Tool | What I can do with it |
|---|---|
| UniProt / Swiss-Prot | Find a reviewed protein record and identify its function, sequence length, domains, GO terms, evidence level, and linked references. I need more practice interpreting detailed evidence sections. |
| NCBI GEO | Review the study design, conditions, sample metadata, biological replicates, and available processed data for a gene-expression study. |
| Count matrices | Understand how RNA-seq counts are organized by gene and sample and why normalization is required before comparison. |
| Differential-expression results | Interpret log2 fold change and adjusted p-values to identify genes that increased or decreased between conditions. |
| Functional enrichment analysis | Interpret overrepresented biological processes in a differentially expressed gene list while recognizing that enrichment does not prove causation. |
| Volcano plots | Read and create a basic volcano plot to identify genes with strong expression changes and statistical significance. |
| Heatmaps | Use a heatmap to compare gene-expression patterns and check whether biological replicates are consistent. |
| InterPro domain analysis | Review predicted protein domains and compare evidence from different domain databases. |
| AlphaFold structure interpretation | View a predicted protein structure and use confidence scores to distinguish more reliable regions from uncertain or flexible regions. |
| RStudio | Open an R project, run provided scripts, inspect objects, and use the Console, Environment, Files, and Plots panels. |
| R scripting | Read and make guided changes to an R script containing objects, vectors, data frames, filtering, summaries, and plots. |
| CSV data handling | Organize tabular data in a CSV file, import it into R, inspect its structure, and identify incorrect column types. |
| Data cleaning in R | Find missing values, replace incorrect text entries with `NA`, convert columns to the correct type, and calculate summaries while accounting for missing data. |
| Basic R visualization | Create boxplots, histograms, and volcano-style plots from provided datasets and save the figures in an Outputs folder. |
| Capstone statistical analysis in R | I am preparing to import the Pb and Cd results from the Triton X-100 and hexane methods, calculate recovery, compare the methods using a paired t-test when appropriate, and create graphs of the results. |
