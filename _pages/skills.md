_An honest summary of my current bioinformatics competencies._

## Intermediate / Advanced

_Skills I can perform independently, explain to others, and apply in a new situation._

| Skill / Tool | What I can do with it |
|---|---|
| BLASTN sequence searches | Prepare a nucleotide FASTA query, select an appropriate NCBI database such as nr/nt, run a BLASTN search, and interpret the main results. |
| BLAST result interpretation | Evaluate sequence matches using E-value, percent identity, query coverage, and bit score, and distinguish full-length matches from short partial alignments. |
| NCBI Gene | Locate a gene record, confirm the gene name, organism, Gene ID, chromosome location, and identify links to related sequence records. |
| NCBI RefSeq sequence retrieval | Retrieve a specific gene sequence from a reference chromosome, verify the coordinates and strand, and avoid accidentally using the complete chromosome sequence. |
| FASTA sequence handling | Recognize FASTA structure, check the sequence header and length, and prepare nucleotide sequences as input for bioinformatics tools. |
| Database annotation comparison | Compare NCBI and UniProt records for completeness, currency, reliability, consistency, functional information, and supporting evidence. |
| Bioinformatics documentation and reproducibility | Record accession numbers, database names, query lengths, search parameters, access dates, screenshots, and analysis steps so the work can be repeated. |
| Reviewed versus unreviewed records | Distinguish curated resources such as Swiss-Prot and reviewed RefSeq records from less-curated database entries and consider annotation quality when selecting data. |

## Beginner / Basic Exposure

_Skills I have used in a guided setting and would need support to apply independently._

| Skill / Tool | What I can do with it |
|---|---|
| UniProt / Swiss-Prot | Look up a reviewed protein record and identify its protein name, function, sequence length, evidence level, GO terms, and literature links. I would need more practice interpreting detailed evidence sections. |
| BLAST Graphic Summary | Use the visual alignment display to examine where database hits align along a query and identify strong, full-length matches. |
| Comparing BLAST hits | Compare the organisms, accession numbers, alignment statistics, and annotations of top hits. I would need support when interpreting more distant homologs. |
| Functional inference from sequence similarity | Make a cautious initial prediction about gene or protein function from strong BLAST matches while recognizing that sequence similarity alone does not prove identical function. |
| BLAST Distance Tree | Understand that a distance tree provides a visual summary of similarities among BLAST results. I would need more practice interpreting evolutionary relationships from it. |
| Multiple sequence alignment | Understand that multiple sequence alignment can identify conserved and variable positions, but I have limited hands-on experience performing and interpreting an analysis. |
| GO terms and evidence codes | Identify GO terms and evidence information in a protein record, but I would need additional support to interpret the different evidence-code categories in depth. |
| Benchling | Have basic familiarity with the platform and its possible use for sequence management and primer design, but would need guidance to build an independent workflow. |
